THE FONT
Have you ever had an experience of just watching the rain pour? Downpour is an experimental-variable font inspired by the heavy rainfall and how it draws certain shapes. The font is a display font that focuses its visual to be less visible on purpose. It is intended to be used by everyone who likes to watch the rain pour and loves its visuals. On systems that support variable fonts, you can select a specific range for how heavy you want the rainfall. The font borrows visual cues from Geomanist Black.
